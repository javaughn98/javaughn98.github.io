Full-Stack/React Projects

Bookstore Web App

This is a full-stack web application that uses the MEAN stack. This is a bookstore application that allows the user to add, edit and delete books from an REST API that I created and displays the information that was entered on the client side. The books that were added are stored in the database that was created.

Stack: Mongodb, Express, Angular JS (front-end), Node JS (MEAN stack), Javascript, API, HTML, CSS.

Shopping List Web App

This project is a shopping list web application where the user is able to list, add and delete items on the shopping list. The application first sends a request to the API then displays the items that are already stored in the database. A modle will pop up whenever you want to add an item which sends a rquest to the API and adds the item to the database.

Stack: Mongodb, Express, React, Node JS (MERN stack), HTML, CSS, Javascript, Axios (HTTP Client).
